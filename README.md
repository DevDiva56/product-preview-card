## Overview

### The challenge

Users should be able to:

- View the best layout depending on their screen size (mobile or desktop)
- See the product card with image, product information, price, and button
- Experience a responsive layout that works on different screen sizes

The designs were created to the following widths:

- Mobile: **375px**
- Desktop: **1440px**

The layout should also work well across different screen sizes starting from **320px and above**.

### What I learned

While building this project, I learned several important things.

First, I learned how to use **Flexbox** to create a two-column layout. On desktop the image and text appear side by side, and on mobile they stack on top of each other.

I also learned how to **center elements on the page** using Flexbox with `justify-content` and `align-items`.

Another important thing I learned was the difference between **margin and padding**. Padding creates space inside an element, while margin creates space between elements.

I also learned how to control images so they stay inside their container using `width: 100%`.

Finally, I practiced how to make a layout **responsive** using media queries so the card works on both mobile and desktop screens.
